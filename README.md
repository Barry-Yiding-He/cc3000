# CC3000 中文说明

CC3000 是一个使用 C++14 编写的终端地牢探索游戏。玩家在随机生成的五层地牢中移动、拾取金币、使用药水、与敌人战斗，并寻找通往下一层的楼梯。进入第 6 层后游戏胜利；如果生命值降到 0，则游戏失败。

## 编译与运行

项目使用 `Makefile` 构建，默认生成可执行文件 `cc3k+`。

```sh
make
./cc3k+
```

清理编译产物：

```sh
make clean
```

如果 `make clean` 在没有编译过的目录中报找不到文件，可以忽略；先执行一次 `make` 后再清理即可。

## 开始游戏

启动程序后会依次询问：

1. 是否控制随机化：输入 `Yes` 或 `No`。
2. 选择玩家种族：输入 `H`、`D`、`E` 或 `O`。

可选种族：

| 输入 | 种族 | 初始 HP | Atk | Def |
| --- | --- | ---: | ---: | ---: |
| `H` | Human | 1400 | 200 | 20 |
| `D` | Drawf | 100 | 20 | 30 |
| `E` | Elves | 140 | 30 | 10 |
| `O` | Orc | 180 | 30 | 25 |

进入地牢后，`@` 表示玩家当前位置。界面下方会显示当前种族、金币数量、楼层、HP、Atk、Def 和最近行动记录。

## 游戏命令

主循环中可以输入以下命令：

| 命令 | 作用 |
| --- | --- |
| `move` | 移动玩家。程序随后会要求输入方向。 |
| `drink` | 使用相邻格子的药水。程序随后会要求输入方向。 |
| `attack` | 攻击相邻格子的敌人。程序随后会要求输入方向。 |
| `surrender` | 认输并结束当前局。 |
| `restart` | 重新开始一局。 |
| `quit` | 退出游戏。 |

方向输入：

| 输入 | 方向 |
| --- | --- |
| `no` | 北 |
| `so` | 南 |
| `ea` | 东 |
| `we` | 西 |
| `ne` | 东北 |
| `nw` | 西北 |
| `se` | 东南 |
| `sw` | 西南 |

示例：

```text
move
ea
```

表示向东移动一格。

## 地图符号

| 符号 | 含义 |
| --- | --- |
| `@` | 玩家 |
| `.` | 房间地面 |
| `#` | 通道 |
| `+` | 门 |
| `\` | 楼梯 |
| `P` | 药水 |
| `G` | 金币 |
| `C` | 罗盘 |
| `B` | Barrier Suit |
| `V` | Vampire |
| `W` | Werewolf |
| `T` | Troll |
| `N` | Goblin |
| `M` | Merchant |
| `X` | Phoenix |
| `D` | Dragon |

楼梯不会一开始显示在地图上。击败持有罗盘的敌人后，会掉落 `C`；拾取罗盘后，通往下一层的楼梯 `\` 会出现。

## 药水

药水以 `P` 显示，必须站在药水相邻八格内，通过 `drink` 加方向使用。药水类型在使用前未知，效果包括：

| 缩写 | 效果 |
| --- | --- |
| `RH` | 恢复最多 10 点 HP |
| `BA` | Atk 增加 5 |
| `BD` | Def 增加 5 |
| `PH` | 失去最多 10 点 HP |
| `WA` | Atk 减少 5 |
| `WD` | Def 减少 5 |

## 金币与特殊物品

金币以 `G` 显示，玩家移动到金币所在格子即可拾取。金币类型包括普通金币、小堆金币、商人宝藏和龙守宝藏。

有些金币或特殊物品默认不可拾取：

- Dragon 守护的金币需要先击败相邻的 Dragon。
- Barrier Suit 也由 Dragon 守护，需要击败守护 Dragon 后才能拾取。
- Merchant 默认不主动敌对；攻击 Merchant 后会使其变为敌对。

## 战斗

输入 `attack` 后选择方向即可攻击相邻敌人。攻击会根据攻击者 Atk 与防御者 Def 计算伤害。敌人死亡后可能掉落金币或罗盘。

玩家移动和战斗后，附近敌人可能会攻击玩家。HP 降到 0 时游戏失败，并显示本局已收集的金币。

## 项目结构

| 文件 | 说明 |
| --- | --- |
| `main.cc` | 程序入口，处理游戏主循环和用户输入。 |
| `floor.h` / `floor.cc` | 地牢楼层、生成逻辑、移动、拾取、战斗和显示。 |
| `character.h` / `character.cc` | 角色基类、玩家和敌人的通用属性与行为。 |
| `player.h` / `player.cc` | Human、Drawf、Elves、Orc 等玩家种族。 |
| `enemy.h` / `enemy.cc` | Vampire、Werewolf、Troll、Goblin、Merchant、Phoenix、Dragon 等敌人。 |
| `item.h` / `item.cc` | 药水、金币、罗盘和 Barrier Suit。 |
| `chamber.h` / `chamber.cc` | 房间区域与坐标管理。 |
| `stair.h` / `stair.cc` | 楼梯对象。 |
| `map.txt` | 地牢基础地图。 |
| `compass.txt` / `compass2.txt` | 方向提示图。 |
| `win.txt` / `lose.txt` | 胜利和失败相关资源。 |
| `Makefile` | 构建配置。 |

## 开发环境

- 语言标准：C++14
- 编译器：`g++`
- 输出程序：`cc3k+`
